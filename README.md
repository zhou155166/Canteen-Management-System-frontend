# Canteen-Management-System-frontend
食堂管理系统 这是一个基于Django + Vue.js的食堂管理系统，提供个性化菜品推荐功能。  系统特色 个性化菜品推荐：根据用户偏好推荐合适的菜品 情景感知推荐：结合天气、节日、客流等信息优化推荐 多维度筛选：支持按品类、口味、价格等条件筛选菜品 响应式界面：适配各种设备的用户界面 技术栈 后端 Django + Django REST Framework MySQL 数据库 前端 Vue.js 3 Vue Router Vuex 状态管理 Element Plus UI组件库 运行流程 环境要求 Python 3.8+ MySQL 5.7+ Node.js 14+
